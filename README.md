Proyecto Final - Blog Nicolás Godoy
Proyecto final realizado en **Python con Django** para la entrega final.

## Descripción

El blog es una aplicación web estilo blog desarrollada con Django.  
La web cuenta con administración, registro de usuarios, login, logout, perfiles, mensajería entre usuarios y gestión de páginas mediante operaciones CRUD.

## Funcionalidades

- Home
- Vista "Acerca de mí"
- Listado de páginas/posts
- Detalle de cada página
- Creación de páginas
- Edición de páginas
- Eliminación de páginas
- Registro de usuarios
- Inicio y cierre de sesión
- Perfil de usuario
- Edición de perfil
- Cambio de contraseña
- Sistema de mensajería entre usuarios
- Panel de administración de Django

## Tecnologías utilizadas

- Python
- Django
- SQLite
- HTML
- CSS
- CKEditor
- Pillow

## Estructura del proyecto

- `pages/` → app principal del blog
- `accounts/` → app de autenticación y perfiles
- `messaging/` → app de mensajería
- `templates/` → plantillas HTML
- `static/` → archivos estáticos
- `media/` → imágenes subidas por usuarios
